# 🚀 AWS S3 Static Website

## 📌 Project Description

This project demonstrates how to host a simple static website using Amazon S3.
The website is publicly accessible using an S3 website endpoint.

---

## 🛠️ Technologies Used

* HTML
* AWS S3

---

## 📂 Project Structure

* index.html → Main webpage file

---

## 🚀 Steps Performed

1. Created a basic HTML file
2. Created an S3 bucket
3. Uploaded index.html
4. Enabled static website hosting
5. Made the bucket public using bucket policy

---

## 🌐 Live Website Link

http://your-bucket-name.s3-website-<region>.amazonaws.com
